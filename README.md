# ☕ Knowledge Cafe

A modern and responsive blog reading web application built with **React.js**, **Tailwind CSS**, and **Fetch API**. Users can read blogs, bookmark articles, and track reading time with an interactive and clean user interface.

## 🚀 Live Demo

🔗 Live Site: https://knowledge-cafe-pink.vercel.app

## 📂 GitHub Repository

🔗 Repository Link: https://github.com/istewakhassantewak/Knowledge-Cafe

---

## ✨ Features

- 📚 Read and explore blogs dynamically
- 🔖 Bookmark favorite blogs
- ⏱️ Reading time calculation and tracking
- ⚛️ Built with React + Vite
- 🎨 Responsive UI using Tailwind CSS
- 📡 Dynamic data fetching with Fetch API
- 📱 Mobile-friendly design
- 🔥 Clean and modern user experience

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- Tailwind CSS
- HTML5
- Vite
- Fetch API

## 📦 Installation & Setup

Clone the repository and run the project locally.

```bash
# Clone the repository
git clone https://github.com/istewakhassantewak/Knowledge-Cafe.git

# Navigate to project folder
cd Knowledge-Cafe

# Install dependencies
npm install

# Run development server
npm run dev
🖥️ Build for Production
npm run build
